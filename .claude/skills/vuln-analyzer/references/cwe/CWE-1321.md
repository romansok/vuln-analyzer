# CWE-1321: Prototype Pollution

## Plain-English explanation
JavaScript objects inherit from a shared prototype (`Object.prototype`). If user-controlled input is merged into an object via a naive deep-merge or copy, an attacker can write to the special key `__proto__` (or `constructor.prototype`) and add or change properties on the shared prototype — affecting **every** object in the program.

## Typical attack pattern
An API accepts a JSON config and merges it into a default object with a `_.merge(defaults, body)` (older lodash). The attacker sends `{"__proto__":{"isAdmin": true}}`. From then on, every object in the process has `isAdmin === true` by default. An auth middleware that reads `user.isAdmin` now sees `true` for everyone.

## Reachability hints
- `_.merge`, `_.defaultsDeep`, `_.set` (older lodash versions), `merge`, `extend`, hand-rolled deep merges.
- `Object.assign(target, JSON.parse(userInput))` — flat, no recursion, generally safe.
- `JSON.parse(userInput)` itself sets `__proto__` only on the top object; the danger is what code does with it next.
- Express body parsers that hydrate query strings into nested objects: `req.query.foo[__proto__][bar]=…` style.

## Common workarounds when a bump is blocked
- Switch to `Object.create(null)` for any object built from user input — it has no prototype to pollute.
- Sanitize input before merge: strip keys named `__proto__`, `constructor`, `prototype`.
- Use a deep-merge that explicitly skips those keys (modern `lodash.mergeWith` with a customizer, or `defu`).
- Freeze `Object.prototype` at startup: `Object.freeze(Object.prototype)` — defense in depth.

## Concrete code example

```js
// Vulnerable
const _ = require('lodash');     // older version
app.post('/config', (req, res) => {
  const cfg = _.merge({}, defaults, req.body);   // __proto__ poisoning
  applyConfig(cfg);
  res.json({ ok: true });
});

// Safer
function safeMerge(target, src) {
  for (const k of Object.keys(src)) {
    if (k === '__proto__' || k === 'constructor' || k === 'prototype') continue;
    if (typeof src[k] === 'object' && src[k] !== null && typeof target[k] === 'object') {
      safeMerge(target[k], src[k]);
    } else {
      target[k] = src[k];
    }
  }
  return target;
}
```
